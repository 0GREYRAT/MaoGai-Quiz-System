# 毛概高频重难点刷题系统 (C#)

这是一个为期末考试设计的《毛泽东思想和中国特色社会主义理论体系概论》复习工具。

## 🚀 项目功能
- [cite_start]**考点精准**：涵盖马克思主义中国化、新民主主义革命、社会主义改造等 103 道核心题目 
- **强制记忆**：采用循环纠错机制，答错必须重答，加深记忆。
- **简单易用**：基于 C# Console 开发，完美适配 Visual Studio 2022。


## 🛠️ 如何运行
1. 克隆本项目或下载 `Program.cs` 文件。
2. 在 Visual Studio 2022 中创建新的 C# 控制台应用。
3. 替换 `Program.cs` 代码并按下 `F5` 运行。
